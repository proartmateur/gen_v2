Tool for write code faster with your own architecture.

## For Contributors

### Tech Stack

In order to modify the CLI you need install:

- Rust v1.81.0+

**Run**

```bash
cargo run
```

**Build Debug**

```bash
cargo build
```

**Build Release**

```bash
cargo build --release
```
